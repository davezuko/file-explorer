## Get Started

Requirements:

-   node @ ^10.18.0
-   yarn ^1.10.1

```sh
# install dependencies
yarn install

# start dev server at http://localhost:3000
yarn start

# build to disk for production
yarn build

# serve the production build locally at http://localhost:3000
yarn serve
```
